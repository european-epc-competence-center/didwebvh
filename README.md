# did:webvh Java Library

A Java library implementing the [`did:webvh`](https://identity.foundation/didwebvh/v1.0/) DID method (DID Web + Verifiable History). `did:webvh` enhances `did:web` with cryptographic verifiability, a tamper-evident history, self-certifying identifiers (SCIDs), and optional features such as key pre-rotation, witnesses, and DID portability — without relying on a ledger.
